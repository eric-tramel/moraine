# Bundled webfonts

These `.woff2` files are self-hosted so the landing page makes **no external
network requests** (privacy-friendly, and faithful to Moraine's local-first
ethos). Both families are licensed under the SIL Open Font License 1.1.

| Family | Weights | License | Source |
| --- | --- | --- | --- |
| Space Grotesk | 500, 600, 700 | OFL-1.1 | <https://github.com/floriankarsten/space-grotesk> |
| JetBrains Mono | 400, 500 | OFL-1.1 | <https://github.com/JetBrains/JetBrainsMono> |

Subset: Latin. Fetched via the Fontsource CDN
(`https://cdn.jsdelivr.net/fontsource/fonts/...`). To refresh, re-run the curl
commands or pull newer weights from Fontsource.
